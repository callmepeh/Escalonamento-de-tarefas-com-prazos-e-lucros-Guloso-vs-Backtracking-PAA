# Escalonamento-de-tarefas-com-prazos-e-lucros-Guloso-vs-Backtracking-PAA
Repositório contendo a implementação e análise comparativa entre duas abordagens de projeto de algoritmos — Estratégia Gulosa e Backtracking (Tentativa e Erro) — aplicadas ao problema de Escalonamento de Tarefas com Prazo e Lucro, incluindo código-fonte, benchmarks, gráficos, apresentação e relatório final.
